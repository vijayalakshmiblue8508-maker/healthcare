**Healthcare Patient Analysis System**


**Overview**


Healthcare analytics is essential for improving patient outcomes and hospital efficiency
This project demonstrates how synthetic patient data can be generated and analyzed using Python
It helps identify trends, patterns, and health risks


**Objective**


тАвManaging patient data manually is time-consuming and inefficient
тАвThis system helps to:
тАвTrack patient records efficiently
тАвIdentify disease patterns
тАвMonitor health parameters
тАвSupport better decision-making


**Approach**


The project uses Python tools:
Data Generation тАУ Using Faker to create realistic data
Data Processing тАУ Using Pandas for cleaning data
Analysis тАУ Applying statistical methods
Visualization тАУ Using charts and graphs


**System Design**


The system has four layers:
Data Generation Layer тАУ Creates synthetic data
Processing Layer тАУ Cleans and structures data
Analysis Layer тАУ Performs analysis
Visualization Layer тАУ Displays insights


**Dataset Details**


Patient_ID тАУ Unique identifier
Name тАУ Generated patient name
Age тАУ Between 20 and 80
Gender тАУ Male or Female
Disease тАУ Type of illness
Sugar_Level тАУ Blood sugar level
Heart_Rate тАУ Beats per minute
City тАУ Location
Admission_Date тАУ Admission date


**Data Analysis Performed**


Age and sugar distribution analysis
Disease frequency analysis
Gender comparison
Admission trend analysis
Outlier detection


**Visual Insights**


Histogram тАУ Age and sugar distribution
Bar Chart тАУ Disease frequency
Pie Chart тАУ Gender ratio
Scatter Plot тАУ Age vs Heart Rate
Box Plot тАУ Disease vs Age
Line Graph тАУ Monthly admissions
Stacked Bar Chart тАУ Gender vs Disease


**Key Findings**


Diabetes is common
Older patients have higher sugar levels
Heart rate varies with age
Some months have more admissions
